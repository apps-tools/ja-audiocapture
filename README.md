# ja-audiocapture
オーディオをキャプチャできるソフト

> [!WARNING]
> **このソフトは個人的に作成したソフトなためサポート対象外です**
>
> .net framework3.5版は推奨されません

## 出力
> [!NOTE]
> .wavと.mp3で出力が可能です

目標出力拡張子
- [x] .mp3
- [x] .wav
- [ ] .aac

> [!WARNING]
> .net framework3.5版では .wavのみ出力可能

## ソースコード

ソフトが安定してきて、コードがきれいになったら配布します
